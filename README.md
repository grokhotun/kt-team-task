# Тестовое задание для kt.team

Проект создан с помощью [Create React App](https://github.com/facebook/create-react-app).


## Задача
Реализовать TODO list: добавление задачи в список, удаление задачи, редактирование задачи
## Требования

 1. Typescript (без any)
 2. В качестве базовых компонентов использовать Ant Design
 3. Покрыть приложение тестами (чем ближе coverage к 100% тем лучше)


## Для запуска
```sh
npm run start
```
или
```sh
yarn start
```
